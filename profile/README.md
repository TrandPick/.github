# 🧥 TrendPick

### 취향 별 추천 플랫폼 TrendPick
태그를 토대로 사용자에게 보다 효율적인 상품 추천을 할 수 있습니다. 사용자가 원하는 스타일, 코디에 맞게 상품들을 추천하여 어떠한 상품을 구매할 것인지 선택할 수 있습니다.
